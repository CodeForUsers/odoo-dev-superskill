To use this module, you need to:

1. Go to *{{ module_menu_path }}*.
2. Click on "Create" to add a new record.
3. Fill in the required fields:

   * **Name**: the name of the record.
   * **Field 2**: description.

4. Click "Confirm" to validate the record.

.. tip::
   You can use the search bar to filter records by status or other criteria.
